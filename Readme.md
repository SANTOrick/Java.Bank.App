How to use
===========
Download the files with Eclipse.  
Go to File > Import > Git > Project from Git.  
Select Clone URI, then in the URI: field paste ```https://github.com/SANTOrick/Java.Bank.App``` and save it in your desired folder.  
Run the body.java
